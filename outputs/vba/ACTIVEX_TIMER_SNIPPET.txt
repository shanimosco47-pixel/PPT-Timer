Slide or Slide Master ActiveX Timer event example:

Private Sub tmrCountdown_Timer()
    PollTimerTick
End Sub

Suggested control settings:
- Name: tmrCountdown
- Interval: 1000
- Enabled: True only while slideshow timer is active

Recommended flow:
1. Start the slideshow timer with:
   StartCountdownTimer 300
2. Let the ActiveX timer call PollTimerTick every second.
3. Stop manually if needed with:
   StopCountdownTimer
